# Quarto slides template

A Quarto RevealJS slides template that runs on a Docker container.

## Usage

### Preview

You can use *Docker* to **preview** the slides by running the following command:

```bash
docker compose up --build
```

### Publish

You can **publish** your slides using *Docker* and *Github Pages*. You will need Git's *username and email* configured, and an SSH key linked to your Github account.

You can then publish it by running the `publish.sh` script.
